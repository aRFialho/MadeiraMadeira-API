5. Create Static Site for Frontend:
   - Build: npm install-all && npm -w frontend run build
   - Publish: frontend/dist

Authentication:
  POST   /api/auth/register          (create account)
  POST   /api/auth/login             (login)

Products:
  GET    /api/marketplace/products   (list products)
  PUT    /api/marketplace/products/:sku (edit product)

Orders:
  GET    /api/marketplace/orders     (list orders)
  GET    /api/marketplace/orders/:orderId/detail (order details)
  PUT    /api/marketplace/orders/:orderId/status (update status)

Dashboard:
  GET    /api/marketplace/dashboard/metrics (metrics)
  GET    /api/marketplace/dashboard/chart (sales chart)

System:
  GET    /health                     (health check)
  GET    /                           (API info)
